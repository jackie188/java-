

## 开发文档结构说明

### 选用技术以及框架版本

本项目是基于Flink的大数据实时数据仓库项目，使用的语言版本和技术框架版本说明：

- Flink：1.12.0
- Scala：2.12
- java:1.8
- hadoop:3.1.3
- fastjson：1.2.68
- mysql-connector-java：5.1.49
- flink-connector-mysql-cdc：1.2.0
- slf4j-api：1.7.25
- clickhouse-jdbc:0.2.4
